# Sample Banking Smart Contract

This project includes a banking smart contract with basic banking:
- Account creation
- Deposit
- Withdrawal
- Transfer(In Bank)
- Transfer(On-Chain)

---

Contract Address: 0xb3a7652cfBaaBb989cA55E5e680110Be00Af73aA

---


Environment Variables

- INFURA_API_KEY
- SEPOLIA_PRIVATE_KEY
- ETHERSCAN_API_KEY


Build

```shell
npm run build
```


Test

```shell
npm run test
```


Deploy

```shell
npm run deploy:Bank
```
