# Sample Banking Smart Contract

This project includes a banking smart contract with basic banking:
- Account creation
- Deposit
- Withdrawal
- Transfer(In Bank)
- Transfer(On-Chain)

---

Contract Address: 0x2BBA18367A7028fA5544B159C626daf54dcFecC6

---


Environment Variables

- INFURA_API_KEY
- SEPOLIA_PRIVATE_KEY
- ETHERSCAN_API_KEY


Build

```shell
npm run build
```


Test

```shell
npm run test
```


Deploy

```shell
npm run deploy:Bank
```
